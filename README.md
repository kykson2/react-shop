### 설명
상품들을 장바구니에 담을 때 redux를 사용해 전역으로 관리할 수 있게 했고

localstorage에도 장바구니 목록을 저장하여 페이지에 재접속했을 때 기존 장바구니 목록을 불러옵니다.


---
### 시작
```
// step 1 관련 패키지 설치
npm init

// step 2 실행
npm run dev

```
